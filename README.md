# Tribute to Dr. A.P.J. Abdul Kalam

An interactive tribute website built with **HTML, CSS, and JavaScript** to honor Dr. A.P.J. Abdul Kalam — the "Missile Man of India" and 11th President of India.  
This project showcases his biography, achievements, inspirational quotes, and fun facts in a modern, engaging layout.

---

## 🌐 Live Demo
[View Tribute Page]https://github.com/yellahansika/TributePage-Dr.Kalam

---

## ✨ Features
- **Hero Section**: Portrait of Dr. Kalam with rotating inspirational quotes.
- **Biography**: Overview of his life, career, and contributions.
- **Achievements**: Bullet-point list of his major milestones.
- **Interactive Section**: Clickable button to reveal random fun facts.
- **Blue Theme**: Clean, professional design with responsive layout.
- **Footer**: Credits the creator.
